# John Rambo: Space Shooting Game 
A 2D arcade-style C++ console game developed for the **Programming Fundamentals Lab** at **FAST-NUCES**.

## Overview
This project is an engaging console adventure where players control "Rambo" in a spaceship. It was designed to demonstrate core programming logic, real-time input handling, and persistent data storage.

## Features
* **Dynamic Gameplay:** Auto-scrolling environment with randomly generated obstacles and enemy tanks.
* **Progressive Difficulty:** Two-level system where speed increases as you advance.
* **Score System:** Persistent high-score tracking saved to `highscores.txt`.
* **Visuals:** 50x30 ASCII character display with color-coded elements.
